## Build
*NPM request 需要npm包管理器*  
主要配置文件在：/src/public/static/config.js  
**注意：请务必配置完成在构建**  

```
npm run build
```
## Install
*请确认配置正确*
将/dist/目录下的所有文件拷贝到Web服务器中,index.html为主页
